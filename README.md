# AI Philosophy Salon
University Of Nottingham

This project is part of my final year work supervised by Dr Pual Tennent.  
It is a performative AI system where four philosopher agents (Aristotle, Sartre, Wittgenstein, and Russell) discuss philosophical topics using local LLMs.

## Structure
- **backend/** — Python backend and model scripts
- **unity/** — Unity front-end (TTS, animation)
- **docs/** — Diagrams and progress notes
